# yrb-redis

> A Redis companion for y-rb

## Installation

Add this line to your application's Gemfile:

```ruby
gem "y-rb_redis"
```

And then execute:

```bash
$ bundle
```

Or install it yourself as:

```bash
$ gem install y-rb_redis
```
